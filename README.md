# Veille & Modération – Violences sexuelles numériques

Ce dépôt contient le code source du site **GitHub Pages** de **Potite_Bulle**,  
dédié à la veille bénévole et à la documentation sur les violences sexuelles numériques en ligne.

Le site est accessible ici (une fois GitHub Pages activé) :  
https://PotiteBulle.github.io

---

## Objectifs du site

- Documenter les schémas de prédation et de violences sexuelles numériques.
- Partager un retour d’expérience de modération bénévole.
- Proposer des ressources de prévention et de signalement.
- Mettre en lumière les limites des plateformes, **sans exposer directement les victimes ni les auteur·ices**.

Aucun contenu explicite ou illégal n’est hébergé dans ce dépôt ou sur le site.

---

## Structure du dépôt

Structure prévue (évolutive) :

- `index.md`  
  Page d’accueil du site : introduction, trigger warning, cadre éthique, informations générales.

- `pages/` (à venir)  
  Pages thématiques : schémas de serveurs, journal de veille anonymisé, ressources, etc.
